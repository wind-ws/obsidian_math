

**定义** 以点 $a$ 为中心的任何开区间，都称为点 $a$ 的邻域，记为 $U(a)$ 。

对任意 $\delta >0$ , 开区间 $(a-\delta, a+\delta)$ 称为 $a$ 的 $\delta$ 邻域，记为 $U(a,\delta)$ ，其另一写法为：$U(a, \delta) = \{x: |x-a| < \delta\}$

若去掉点 $a$ , 即 $\stackrel{\circ}{U}(a, \delta) = \{x: 0 < |x-a| < \delta\}$ ，称为点 $a$ 的去心 $\delta$ 邻域

**注**：邻域的直观理解，就是邻近的区域。通过取任意小的 $\delta$ 值，就可以用 $U(a,\delta)$ 来表示接近 $a$ 的程度。



### **无穷的邻域**

$U(+\infty)=U(+\infty, M) = \{x: x > M \}$

$U(-\infty) = U(-\infty, M) = \{x: x < -M \}$

![[附件/v2-7d173ec22c9d366df140df7dd9ef1879_720w.webp]]

$U(\infty) = U(\infty, M) = \{x: |x| > M \}$

![[附件/v2-e6ed3b55e5fa767b86c59d2f2425a9ae_720w.webp]]

其中，$M$ 取足够大的正数即可。

### **单侧邻域**

$U(x_0^-, \delta) = \{x: 0 < x_0 - x < \delta\}$
